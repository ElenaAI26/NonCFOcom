TECHNICAL REQUIREMENTS

Single HTML file OR React/Vite single page — whatever runs cleanly in Replit
Mobile-first responsive design
Smooth scroll between sections
Sticky nav that changes style on scroll (JS scroll listener)
Hamburger menu on mobile
Form submission shows thank-you state without page reload
Hover effects on cards (translateY lift + shadow)
No external dependencies beyond Google Fonts and one icon library (use Lucide CDN or simple SVG icons inline)
All sections clearly ID'd for anchor links (#services, #contact, etc.)


COPY RULES

Write like a premium consulting firm, not a generic nonprofit
Avoid clichés like "make a difference" or "change the world"
Use "mission-driven" sparingly and purposefully
Keep CTAs action-oriented and specific
